처음이어도 괜찮아요

자녀가 한 번만 등록하면
부모님은 링크 하나로 계속 편하게 주문!

자녀(관리자)가 가게·메뉴·주소를 등록한 뒤 링크를 만들어 부모님께 전달해요. 부모님은 그 링크만 열면 로그인 없이 바로 주문할 수 있어요.

1
접속 & 구글 로그인
앱을 Safari 또는 Chrome으로 열고 "Google로 시작하기"를 누르세요.
⚠️카카오톡 안에서 링크를 눌렀다면 구글 로그인이 막혀요. 우측 상단 ⋯ → 다른 브라우저로 열기로 Safari/Chrome에서 다시 열어주세요.
2
가게 추가하기
로그인하면 단골집 관리 화면이 열려요. "＋ 가게 추가하기"를 누르세요.
3
가게 정보 입력
🏠 가게 이름, 📞 전화번호, 🍽 메뉴를 입력해요. 메뉴는 하나씩 입력 후 ＋로 추가하고, 다 됐으면 저장하기를 누르세요.
💡가게·메뉴 이름에 "치킨", "피자", "죽" 같은 단어가 들어가면 아이콘이 자동으로 바뀌어요.
4
우리집 주소 등록 & 링크 공유
① "🏠 우리집 주소" 칸에 배달받을 주소를 입력하고 저장 (배달 멘트에 자동 포함돼요)
② "📋 링크 복사" 또는 "💬 카톡으로 보내기"로 부모님께 주문 링크를 전달!
🔒링크를 받은 분은 주문 화면만 볼 수 있고 수정은 불가능해요. 가게·주소를 바꾸려면 언제든 다시 로그인해서 수정하면 돼요.
5
부모님 주문 실시간 확인
부모님이 주문하면, 자녀 홈 화면 아래 "🔔 부모님 주문 현황"에 어떤 가게·메뉴를 시키셨는지 실시간으로 표시돼요. "🔔 알림 켜기"를 눌러두면 앱이 열려 있을 때 브라우저 알림으로도 바로 받아볼 수 있어요.
🔔부모님이 무엇을 드시는지 멀리서도 확인할 수 있어 안심돼요.
⚠️실제 배포 시에는 주문 알림용 보안 규칙을 적용해야 부모님 주문이 자녀 화면에 기록돼요. (자세한 내용은 README 참고)

자녀에게 받은 링크를 누르면 바로 시작! 회원가입·로그인 필요 없어요.

1
가게 고르기
"어디서 시킬까요?" 화면에서 원하는 가게를 손가락으로 한 번 톡 누르세요.
2
메뉴 고르기
"무엇을 드실까요?" 화면에서 드시고 싶은 메뉴를 누르세요. 여러 개 골라도 되고, 다시 누르면 취소돼요.
3
포장 / 배달 고르기
메뉴 아래에서 받는 방법을 골라요. 🛍️ 포장은 직접 찾아올 때, 🛵 배달은 집까지 받을 때. 기본은 포장이며, 배달은 눌러서 주황색이 되는지 확인하세요.
4
전화 주문하기
주황색 "📞 전화 주문하기"를 누르면 주문 메모가 나와요. 화면 문장을 그대로 읽으면 되고, "📞 전화 걸기"로 바로 연결!
🛍️ 포장 멘트
🛵 배달 멘트 (주소 자동 포함)
5
다음엔 더 빠르게 — 다시 주문하기
한 번 주문하고 나면, 다음부터 홈 화면 위쪽에 ⭐ 다시 주문하기가 생겨요. 자주·최근 시킨 가게와 메뉴가 카드로 떠서, 카드 한 번만 누르면 메뉴와 포장/배달까지 자동으로 골라져 바로 전화로 이어져요.
💡늘 같은 걸 시키신다면, 가게·메뉴를 고를 필요 없이 이 카드만 누르면 끝이에요.
6
홈 화면에 추가하기
처음 열면 위쪽에 "홈 화면에 추가하세요" 안내가 떠요. 추가를 누르면 스마트폰 홈 화면에 앱 아이콘이 생겨, 다음부터는 카톡에서 링크를 찾지 않고 아이콘만 톡 눌러 바로 열 수 있어요.
📲아이폰 Safari는 공유 → 홈 화면에 추가로도 등록할 수 있어요.

자주 묻는 질문

Q. 카톡에서 부모님 링크를 눌렀는데 화면이 이상해요⌄
부모님 전용 링크(?fid=...)는 카카오톡 안에서도 정상 동작해요. 만약 로그인 화면이 보인다면 잘못된 링크이니 자녀에게 다시 받아주세요.
Q. 배달 멘트에 주소가 안 보여요⌄
자녀가 주소를 입력 후 저장 버튼까지 눌렀는지 확인하고, 부모님 화면을 새로고침하면 반영돼요.
Q. 가게를 수정하거나 삭제하고 싶어요⌄
자녀가 관리자 화면(로그인 필요)에서 가게 카드의 ✏️ 수정 / 🗑 삭제 버튼을 누르면 돼요.
Q. 부모님이 링크를 매번 찾기 번거로워요⌄
부모님 화면 위쪽 "홈 화면에 추가하세요 → 추가"를 누르면 홈 화면에 앱 아이콘이 생겨, 다음부터 링크를 찾을 필요 없이 아이콘만 누르면 돼요. 아이폰 Safari는 공유 → 홈 화면에 추가로도 등록할 수 있어요.
Q. 늘 같은 메뉴를 매번 고르기 번거로워요⌄
한 번 주문하고 나면 부모님 홈 위쪽에 ⭐ 다시 주문하기 카드가 생겨요. 자주·최근 시킨 가게와 메뉴가 떠서 한 번만 누르면 메뉴·포장/배달까지 자동으로 골라져 바로 전화 주문으로 이어집니다.
Q. 부모님이 주문하신 걸 알 수 있나요?⌄
네. 자녀 홈 화면 아래 "🔔 부모님 주문 현황"에 주문이 실시간으로 쌓이고, "알림 켜기"를 눌러두면 앱이 열려 있을 때 브라우저 알림으로도 받아볼 수 있어요. (부모님은 로그인하지 않으므로, 실제 배포 시 orders 보안 규칙을 적용해야 알림이 동작해요.)
