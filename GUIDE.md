# 📖 사용 가이드

`단골집 주문` 앱을 처음 쓰는 분도 바로 따라 할 수 있도록 정리한 사용 설명서입니다.

- 접속 주소: **https://qkrxodud.github.io/parents-order/**
- 자녀(관리자)가 가게/메뉴/주소를 등록 → 링크를 만들어 부모님께 전달
- 부모님은 그 링크만 열면 **로그인 없이** 바로 주문할 수 있어요

---

## 1. 자녀(관리자) 가이드

### 1-1. 처음 접속하기

1. 위 접속 주소를 브라우저(카카오톡 인앱 브라우저는 ⚠️ 구글 로그인이 막혀 있어요. **Safari/Chrome**으로 열어주세요)에서 엽니다.
2. **"Google로 시작하기"** 버튼을 눌러 본인의 구글 계정으로 로그인합니다.
3. 로그인하면 자동으로 나만의 "가족방"이 만들어지고, 단골집 관리 화면으로 이동합니다.

> 💡 카카오톡 등 앱 내부 브라우저로 열었다면 안내 문구가 표시됩니다. 우측 상단 **⋯ 메뉴 → 다른 브라우저로 열기**를 눌러 다시 시도해주세요.

### 1-2. 단골 가게 등록하기

1. 단골집 관리 화면에서 **"＋ 가게 추가하기"**를 누릅니다.
2. 아래 정보를 입력합니다.
   - 🏠 가게 이름 (예: `단골 짜장면집`)
   - 📞 전화번호 (예: `02-1234-5678`)
   - 🍽 자주 시키는 메뉴 (메뉴를 하나씩 입력 후 추가 — 여러 개 등록 가능)
3. **저장**을 누르면 가게 카드가 목록에 표시됩니다.
4. 같은 방법으로 부모님이 자주 시키는 가게를 원하는 만큼 등록하세요.
5. 가게 정보를 바꾸고 싶으면 카드의 **✏️ 수정**, 삭제하려면 **🗑 삭제**를 누르면 됩니다.

> 음식 종류에 따라 🍗🍕🍙🥣 등 아이콘이 자동으로 붙어요. 가게/메뉴 이름에 "치킨", "피자", "죽" 같은 단어가 들어가면 알아서 매칭됩니다.

### 1-3. 우리집 주소 등록하기 (배달 주문용)

1. 가게 목록 아래 **"🏠 우리집 주소"** 입력칸에 배달 받을 주소를 입력합니다.
2. **저장** 버튼을 누르면 "주소가 저장되었습니다" 안내가 뜹니다.
3. 이 주소는 부모님이 **배달**을 선택했을 때 주문 메모에 자동으로 표시됩니다.
   (포장을 선택하면 주소는 표시되지 않아요.)

### 1-4. 부모님께 링크 보내기

1. 화면 하단 **"📮 부모님께 보낼 링크"** 영역에서 나만의 주문 링크를 확인합니다.
   - 형태: `https://qkrxodud.github.io/parents-order/?fid=내고유ID`
2. **"📋 링크 복사"**를 눌러 클립보드에 복사하거나, **"💬 카톡으로 보내기"**를 눌러 바로 카카오톡 공유창을 엽니다.
3. 이 링크를 부모님 휴대폰의 카카오톡 등으로 전달하세요.

> 🔒 이 링크는 부모님 전용 "주문 화면"만 보여주고, 가게/메뉴를 수정할 수는 없어요. 등록·수정은 항상 1-1의 관리자 로그인 화면에서만 가능합니다.

### 1-5. 가게/메뉴/주소를 나중에 또 바꾸고 싶다면?

언제든 1-1 주소로 다시 접속해 같은 구글 계정으로 로그인하면, 등록했던 모든 정보가 그대로 남아있습니다. 자유롭게 수정·추가·삭제하세요. 부모님이 가지고 있는 링크는 그대로 유효합니다.

## 2. 부모님 가이드 (링크로 처음 접속)

자녀에게 받은 링크(`...?fid=...`)를 누르면 됩니다. **회원가입이나 로그인은 전혀 필요 없어요.**

### 2-1. 가게 고르기

화면을 열면 **"어디서 시킬까요?"**라는 제목과 함께 등록된 단골 가게들이 큼직한 목록으로 보입니다.

- 원하는 가게를 손가락으로 한 번 톡 누르세요.

### 2-2. 메뉴 고르기

가게를 누르면 **"무엇을 드실까요?"** 화면으로 이동합니다.

- 평소 즐겨 드시던 메뉴들이 보입니다. 원하는 메뉴를 누르면 ✅ 표시가 됩니다.
- 여러 개 선택해도 괜찮아요. 다시 누르면 선택이 취소됩니다.

### 2-3. 포장 / 배달 고르기

메뉴 아래 **"어떻게 받으실까요?"**에서 두 가지 중 하나를 고릅니다.

| 버튼 | 의미 |
|------|------|
| 🛍️ 포장 | 직접 가게에 가서 찾아올 때 |
| 🛵 배달 | 집까지 배달받을 때 |

- 기본값은 **포장**으로 되어 있어요. 배달을 원하시면 **🛵 배달**을 눌러 주황색으로 선택되는지 확인하세요.

### 2-4. 전화로 주문하기

1. 메뉴와 받는 방법을 고르면 하단 주황색 **"📞 전화 주문하기"** 버튼이 활성화됩니다.
2. 버튼을 누르면 **"주문 메모"** 화면이 나옵니다. 여기엔 가게 전화번호와, 전화로 그대로 읽으면 되는 문장이 적혀 있어요.
   - 예: `짜장면, 짬뽕 배달로 주세요.` + (배달일 경우) `주소는 OOO입니다.`
3. **"📞 전화 걸기"**를 누르면 바로 전화 연결이 됩니다.
4. 화면에 적힌 문장을 그대로 읽어 말씀하시면 끝!

> 💬 화면에 표시된 주소가 잘못됐거나 안 보이면, 자녀에게 "우리집 주소를 등록/수정해줘"라고 말씀해주세요. (위 1-3 참고)

### 2-5. 다른 가게로 다시 주문하고 싶을 때

주문 메모 화면 왼쪽 위 **‹ 뒤로** 버튼을 누르면 메뉴 선택 화면으로, 한 번 더 누르면 가게 목록으로 돌아갈 수 있습니다.

## 3. 자주 묻는 질문 (FAQ)

**Q. 부모님이 받은 링크를 다른 사람에게 전달해도 되나요?**
A. 링크가 있으면 누구나 그 가족의 가게 목록과 주소를 볼 수 있어요(주문 화면만, 수정 불가). 가족 외 외부인에게는 전달하지 않는 것을 권장합니다.

**Q. 링크를 열었는데 "등록된 가게가 없어요"라고 나와요.**
A. 자녀가 아직 가게를 등록하지 않은 상태입니다. 자녀에게 1-2 단계로 가게를 등록해달라고 요청하세요.

**Q. 우리집 주소를 등록했는데 배달 메모에 안 보여요.**
A. 자녀가 1-3에서 주소를 입력 후 **저장** 버튼까지 눌렀는지 확인해주세요. 저장 후에는 부모님 화면을 새로고침하면 반영됩니다.

**Q. 카카오톡에서 링크를 눌렀는데 화면이 이상해요.**
A. 부모님 전용 주문 화면(`?fid=...` 링크)은 카카오톡 안에서도 정상적으로 동작합니다. 만약 로그인 화면(자녀용)이 보인다면 잘못된 링크일 수 있으니, 자녀에게 받은 링크가 `?fid=`로 끝나는지 다시 확인해주세요.
